# React Mini Shop

A simple e-commerce project built with React, showcasing basic routing, component composition, and Material-UI implementation.

## Features

- Home page with project introduction
- Products page displaying product cards with state management
- Details page for individual products using React Router
- Clean and modern UI using Material-UI components
- Responsive design for all screen sizes
- Icon integration in buttons and UI elements

## Technologies Used

- React
- React Router DOM
- Material-UI (MUI)
- Vite

## Getting Started

1. Clone the repository
2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

## Project Structure

- `/src/components/` - React components
  - `Home.jsx` - Landing page
  - `Products.jsx` - Product listing with cards
  - `Details.jsx` - Individual product details
- `/src/App.jsx` - Main application component with routing
- `/src/App.css` - Global styles
